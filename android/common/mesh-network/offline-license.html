Offline Only

info

This guide is for syncronizing data entirely offline without ever synchronizing without ever connecting to the Big Peer.

Most applications should instead use Online With Authentication or Online Playground.

Offline license tokens are required to use OfflinePlayground, Manual, and SharedKey identities.

Request a license​

To request a license, go to your app in the Ditto portal and find the Offline License Tokens section.

Offline Playground​

caution

Devices using Offline Playground will not synchronize with devices using Online Playground or Online with Authentication.

Offline Playground are suitable for testing offline-only behavior.

  • Apps with Offline Playground can read and write to all collections.
  • Apps with Offline Playground will connect to other peer devices with the same appID and offlineOnlyLicenseToken.
  • Apps with Offline Playground will not synchronize with the Big Peer.
var ditto = Ditto(identity: DittoIdentity.offlinePlayground(    appID: "REPLACE_ME_WITH_YOUR_APP_ID"))try! ditto.setOfflineOnlyLicenseToken(validLicense);try! ditto.startSync()
